## Button

按钮用于开始一个即时操作。

### 何时使用

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

### Props
|name|type|default|description|
|---|---|---|---|
|type|String|'primary'|button type, `primary` `secondary` `dashed` or `danger`|
|size|String|'normal'|button size, `normal` `large` or `small` |
