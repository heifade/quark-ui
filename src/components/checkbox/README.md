## Checkbox

Checkbox Component.

### Props
#### Checkbox
|name|type|default|description|
|---|---|---|---|
|checked|boolean|false|指定当前是否选中|
|defaultChecked|boolean|false|初始是否选中|
|onChange|Function(e:Event)|-|变化时回调函数|


#### CheckboxGroup 
|name|type|default|description|
|---|---|:---:|:---:|
|defaultValue|Array|[]|默认选中的选项|
|value|Array|[]|指定选中的选项|
|options|Array|[]|指定可选项|
|onChange|Function|-|变化时回调函数|
### Api
