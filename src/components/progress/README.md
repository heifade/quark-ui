## Progress

Progress Component.

### Props
|name|type|default|description|
|---|---|---|---|
|status|string|normal|`normal` `exception` `pause` or `success`|
|percent|number|0|进度百分比|
|showInfo|true|boolean|是否显示进度数值或状态图标|
|size|string|normal|进度条尺寸，`normal` or `mini`|

### Api
